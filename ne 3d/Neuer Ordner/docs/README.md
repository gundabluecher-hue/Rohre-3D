# Dokumentationsstruktur

## Ordner

- `docs/guides`: Anleitungen und Walkthroughs.
- `docs/plans`: Umsetzungs- und Architekturplaene.
- `docs/prompts`: Prompt-Vorlagen fuer Folge-Chats.
- `docs/reports`: Auswertungen und Berichte.

## Hinweis

Laufzeitkritische Dateien fuer das Spiel bleiben im Projekt-Root (`index.html`, `style.css`, `js/`, `bot-learning-data.json`, `learning_history_compact_summary.json`).
